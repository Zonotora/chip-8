# chip-8
A simple emulator for CHIP-8 written in Rust.

![image](images/pong.png?raw=true)


### Usage
The keyboard is mapped to CHIP-8 in the following way:

###### Keyboard
|   |   |   |   |
| - | - | - | - |
| 1 | 2 | 3 | 4 |
| Q | W | E | R |
| A | S | D | F |
| Z | X | C | V |
###### CHIP-8
|   |   |   |   |
| - | - | - | - |
| 1 | 2 | 3 | C |
| 4 | 5 | 6 | D |
| 7 | 8 | 9 | E |
| A | 0 | B | F |
